About SteamSales

Минималистичный агрегатор скидок Steam. Данные получены через публичный Steam Storefront API (featuredcategories).

  • Периодическое обновление — каждые 10 минут.
  • Плитки в стиле Windows 11: чёрная тема, акценты, карточки.

SteamSales — локальный проект для просмотра скидок.
